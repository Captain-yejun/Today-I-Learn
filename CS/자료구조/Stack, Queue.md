- 선형 자료로 1차원의 형태를 띈다.

- Last in First Out : 후입 선출

- 삽입 연산 : Push

- 삭제 연산 : Pop

- ex) 되돌아가기 (ctrl + z)

- Stack underflow : 비어있는 스택에서 원소를 추출하려고 할 때

- Stack overflow : 스택이 넘치는 경우
